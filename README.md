# Mario Jump 🍄

Este é um projeto simples desenvolvido para fins académicos, inspirado no famoso "Dino Run" (o jogo do dinossauro do Google Chrome). O objetivo foi recriar a mecânica de desvio de obstáculos, mas com uma temática personalizada do universo **Super Mario**.

## 🕹️ Sobre o Projeto

O **Mario Jump** é um jogo de plataforma minimalista onde o jogador controla o Mario. O objetivo é saltar sobre os canos que surgem no caminho para evitar a colisão. Conforme o tempo passa, o desafio mantém-se em testar os reflexos do jogador.

Este projeto foi desenvolvido utilizando tecnologias web fundamentais, focando na manipulação do DOM e animações via CSS.

## 🚀 Tecnologias Utilizadas

  * **HTML5:** Estrutura básica do jogo.
  * **CSS3:** Responsável pelas animações do cenário, do Mario e dos obstáculos.
  * **JavaScript:** Lógica do jogo, deteção de colisões e comandos de salto.

## 🎮 Como Jogar

1.  Clona o repositório ou faz o download dos ficheiros.
2.  Abre o ficheiro `index.html` em qualquer navegador.
3.  Pressiona qualquer tecla (ou clica no ecrã) para fazer o Mario saltar.
4.  Evita que o Mario toque nos canos verdes\!

## 🛠️ Estrutura de Ficheiros

```text
MarioJump/
├── index.html      # Estrutura do jogo
├── style.css       # Estilização e animações
└── script.js      # Lógica e mecânicas
```
